# 🚀 Karo Pitch - True 3D Cinematic Landing Page

Karo Pitch is a premium, high-converting landing page built for India's startup founders from Tier-2, Tier-3 cities, MSMEs, and manufacturing sectors. It features a stunning **"True 3D" Spiderman-tier cinematic scroll experience**, designed to captivate top-tier investors and stand out with a Silicon Valley level of web design aesthetics.

## ✨ High-End Cinematic Features

- **Deep Z-Axis Parallax Pinning:** The hero section uses advanced GSAP ScrollTrigger pinning. Instead of just scrolling down, you are pulled *through* the screen into the next section as the background massively scales up.
- **Magnetic Custom Cursor:** Replaces the default pointer with a smooth, emerald-green trailing cursor that magnetically snaps and expands over interactive elements.
- **Live 3D Mouse Tracking:** Glassmorphism UI panels dynamically tilt in 3D perspective (`rotationX`, `rotationY`) by tracking your live mouse movements across the screen.
- **Buttery Smooth Scroll:** Powered by Lenis for an ultra-premium, frictionless scrolling experience that synchronizes perfectly with GSAP timelines.
- **Cinematic Film Grain:** A subtle, fixed SVG noise overlay runs across the entire viewport to simulate a high-end cinematic camera film grain over the deep navy gradients.
- **Elastic VC Logo Popups:** The "Meet Investors" section features a dramatic, bouncy CSS pop-up animation revealing VC firm logos directly over investor avatars on hover.
- **Rich Bento Grid Photography:** Clean, modern "Who Can Apply" bento layouts utilize high-quality Unsplash photography overlays.

## 🛠️ Tech Stack

- **Vanilla HTML5 & CSS3** (Zero heavy CSS frameworks, utilizing custom variables and ultra-fast hardware-accelerated transforms)
- **Vanilla JavaScript**
- **[GSAP (GreenSock) Core](https://gsap.com/) & ScrollTrigger** for complex, performant scroll animations.
- **[Lenis](https://lenis.studiofreight.com/)** for native-feeling smooth scrolling.
- **Phosphor Icons** for sharp, modern iconography.

## 🚀 Getting Started

To run the cinematic demo locally and explore the 3D effects:

1. Clone or download this repository.
2. Open the project folder in your terminal.
3. Start a local HTTP server (highly recommended to ensure smooth execution of local scripts):
   ```bash
   python -m http.server 8000
   ```
4. Open your browser and navigate to `http://localhost:8000`.
